# OSHI RADAR (PORTFOLIO)

JSL人材開発院 Cloud連携Web開発者課程開発したポートフォリオプロジェクトです。 Java, JSP/Servlet, Spring, MyBatis, Oracle DB を用いて構築されており、バックエンド中心のウェブ開発に重点を置いています。

## ✨ プロジェクト紹介
本プロジェクトは、コミュニティ機能・商品管理・位置情報要素を一つのWebサービスとして統合することを目的に企画しました。
サーバーサイドの設計、データフロー、MVC構造の理解と実装経験を深めるため、バックエンド中心で開発を行いました。
Java系Web技術を用いた実践的なバックエンド開発力を示すポートフォリオ作品です。

## 📆 プロジェクト期間
2026.01.10(土) ~ 2026.01.28（水）

## 💻 開発環境
Language    : Java 17

IDE         : Eclipse

Web Server  : Apache Tomcat 9

Framework  : JSP / Servlet

Persistence: MyBatis

Database   : Oracle Database

Frontend   : HTML, CSS, JavaScript

Versioning : Git / GitHub

## 🔧 主な機能の要約
#### ✧ 会員登録およびログイン機能
- Sessionベースの認証処理
- ログインユーザーの権限分岐

#### ✧ 掲示板機能
- 投稿作成 / 参照 / 編集 / 削除
- 画像ファイルアップロードおよびサーバー保存
- タイトル・内容に基づく検索およびページング処理

#### ✧ コメント機能
- 投稿ごとのコメント作成
- 作成者本人のみ削除可能

#### ✧ フリマ機能
- 商品登録および画像アップロード
- 商品リストおよび詳細表示

#### ✧ 共通UI構成
- CSSベースの画面構成およびレスポンシブ対応
